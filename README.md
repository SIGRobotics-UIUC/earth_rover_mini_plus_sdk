# earth_rover_mini_plus_sdk

API for the Earth Rover Mini+. Uploaded to PyPi, installable using pip.

## Installation

Windows:
```bash
py -m pip install earth_rover_mini_plus_sdk
```

Unix/MacOS:
```bash
python3 -m pip install earth_rover_mini_plus_sdk
```
